# Turtlebot3-based project descriptions

Turtlebot3 extended with a raspberry pi camera. 
```bash
$> export PROJECT_MODEL=turtlebot3_burger_pi_cam
$> ros2 launch turtlebot3_gazebo projects_empty_world.launch.py
```
